Instructions

This file contains a single table of Zoom meetings. You can edit it with any text editor and add more rows to the table.

When the page is loaded, a simple JavaScript function adds an extra cell to the end of every table row with a link that, when you click on it, should launch Zoom and connect you to the meeting.

I say "should" because there's a chance that it won't work if the URL protocol handlers in your browser/OS aren't set up to forward zoommtg://... URLs to the Zoom application.

You can also add or delete columns from the table. The script will always use the last two cells in each row.

I got tired of entering strings of numbers for recurring meetings, which I did for 2 months, so I spent an hour writing this. If it breaks, you get to keep all the pieces. It makes my life a little easier, but YMMV.

You can edit the file to delete, or comment out, these instructions.
